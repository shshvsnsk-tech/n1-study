# N1 研习室

一个无需服务器、双击即可使用的 N1 单词与语法背诵网站。

## 使用

双击 `打开N1研习室.cmd`，或直接打开 `index.html`，即可离线学习，不需要安装或启动服务器。

学习进度、收藏和错题记录保存在当前浏览器中。建议定期使用词库页的“导出学习记录”备份数据；更换浏览器后可用“导入学习记录”恢复。

## GitHub Pages 发布

仓库中只需公开以下网站文件：

- `index.html`
- `styles.css`
- `app.js`
- `data.js`
- `README.md`
- `.nojekyll`

`.gitignore` 已排除原书 PDF、扫描页、测试工具、浏览器临时目录、自动化进度和预览图片，避免误上传本地资料。

发布步骤：

1. 在 GitHub 新建仓库并上传本项目。
2. 打开仓库的 `Settings` → `Pages`。
3. 在 `Build and deployment` 中选择 `Deploy from a branch`。
4. 选择 `main` 分支和 `/ (root)` 目录并保存。
5. 等待 GitHub 生成 `https://用户名.github.io/仓库名/` 地址。

## iPhone 使用

使用 Safari 打开 GitHub Pages 地址，然后选择“分享”→“添加到主屏幕”。

电脑和 iPhone 的学习记录彼此独立。迁移记录时，先在原设备的词库页导出 JSON，再在另一台设备导入。iOS 可能清理长期未访问网站的本地数据，建议定期导出备份。

## 内容说明

请勿将两本原始 PDF、扫描图片或其他受版权保护的原书文件提交到公开仓库。本项目发布内容仅保留用于个人学习整理的结构化词条、释义、用法与例句。

## 功能

- 单词和语法翻转卡片
- 到期优先的自适应间隔复习
- 每轮 20 张卡片的完成结算与下一轮入口
- 每轮固定 20 题的小测试
- 混合模式固定包含 10 道词汇和 10 道语法，并分别显示得分
- 单词、语法和错题专项测验
- 搜索与分类筛选
- 收藏、错题和掌握状态
- 错题掌握后清除、手动加入今日复习
- 空格翻卡，数字键 1–4 快速评分
- 使用浏览器内置日语语音朗读单词和语法
- 每日目标、连续学习天数和本地进度
- 每日目标可设置为 10、20、30 或 50 条
- 学习记录导出与导入，便于备份或换浏览器
- N1 非听力模拟考试框架：110 分钟计时、自动保存、答题卡、未答计数与跳转、交卷、成绩历史和分项复盘

模考模块当前仅在本地开发，第一回已扩充到 56 道原创题，第二回已扩充到 50 道原创题并完成本地验证，覆盖全部 N1 非听力题型。三套完整原创试卷通过最终验收并获得确认后再发布到 GitHub Pages。

## 数据

学习数据位于 `data.js`，共 400 条：

- N1 单词 312 条
- N1 语法 88 条

内容根据红宝书与蓝宝书的 N1 部分分批整理，每条均包含释义、用法提示、例句、翻译和来源 PDF 页码。
